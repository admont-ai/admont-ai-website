Features

Features for real knowledge work.

Admont AI combines document creation, diagram editing, AI-assisted authoring, hybrid search, and knowledge-base question answering in one workflow. The focus is practical knowledge work across pages, artefacts, and teams.

  • Portable Markdown-based knowledge
  • Visual and source editing
  • AI assistant and AI agent workflows
  • Hybrid search and RAG

What this enables

  • Create and update documents, diagrams, and artefacts in one system.
  • Use AI for both direct edits and more complex multi-page work.
  • Search and question the full knowledge base with better grounding.

Core Features

Built for real documentation and knowledge operations.

Markdown documents

Create and maintain documents in Markdown as the primary working format, keeping content portable, versionable, and easy to reuse outside the application.

Draw.io and Mermaid diagrams

Create technical and operational diagrams alongside documents using Draw.io and Mermaid so visual knowledge lives with the rest of the repository.

Visual and source editing

Edit content visually when speed matters, or work directly in source when precision matters. The same applies across the main document and diagram workflows.

Limited LaTeX support

Include mathematical or structured notation where needed with limited LaTeX support, without turning the product into a specialist publishing tool.

Spreadsheet-aware knowledge base

Read Excel sheets as part of the knowledge base so structured operational data can contribute to search, AI answers, and generated content.

Portable artefacts

Keep documents and related artefacts in forms that can be versioned, reviewed, and accessed with existing tools instead of being trapped in an opaque content store.

Shared team workflow

Work across documents, diagrams, and other artefacts in a shared environment instead of splitting knowledge work across separate niche editors.

Document and artefact permissions

Control which users can access which documents and artefacts so the knowledge base can stay useful without becoming too open.

Permission-aware search and RAG

Search results and AI retrieval only include information the current user is actually allowed to access.

AI Workflows

AI support ranges from direct editing to broader agent work.

AI assistant for everyday work

Create, revise, summarize, and expand documents and artefacts directly with the built-in AI assistant.

AI agent for complex changes

Use an AI agent for more complex editing tasks that span multiple pages, related documents, and supporting artefacts.

Grounded by existing knowledge

AI output can be informed by retrieved knowledge from the existing corpus instead of starting from a blank context.

Search And Answers

Find information and ask questions across the full knowledge base.

Hybrid document search

Find documents through classic full-text search as well as semantic search, so results can match both exact terms and related meaning.

Knowledge-base questions with RAG

Ask questions across the full knowledge base and use retrieval to ground AI responses in the documents and artefacts your team already maintains.

Excel data included in retrieval

Hybrid search can include information from Excel sheets, which makes spreadsheet data available for RAG-based questions and AI content generation alongside normal documents.

One retrieval layer for people and AI

The same retrieval-ready knowledge base supports both human search workflows and AI-assisted question answering.

Better reporting from existing data

Teams can turn data already stored in spreadsheets into summaries, explanations, and generated reports without manually rewriting the same information into another system first.

Admont AI helps teams create, reuse, and operationalize company knowledge.
