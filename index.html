Product

A document engine built around your existing workflow.

Admont AI sits between your repositories, search stack, language models, and authentication layer. You write in markdown, store everything in Git, and add AI and search where they are useful instead of rebuilding your workflow around them.

Features

Core capabilities without the platform weight.

Git-backed editing

Store documents in standard Git repositories with commit history, diffs, and restore paths built in.

Draft workflow

Save non-destructive drafts, then publish with conflict-aware merging when the underlying file changed.

Hybrid search

Combine full-text and semantic search to find the right content faster across your knowledge base.

AI assistance

Ask questions, generate content, or polish existing text using cloud-hosted or self-hosted models.

Granular permissions

Control access at the repository, folder, or file level with path-based permission rules.

MCP integration

Expose the knowledge base to AI assistants through the Model Context Protocol with authenticated access.

How It Works

Connect storage, search, models, and auth.

Admont AI keeps the browser experience simple while integrating with the systems that matter underneath.

Storage

Git repositories on local, remote, or S3-compatible backends.

Search

Hybrid full-text and vector search with pluggable engines.

Models

Use OpenAI, Anthropic, Google, Ollama, and other providers.

Access

OAuth-based authentication and file-level authorization.
