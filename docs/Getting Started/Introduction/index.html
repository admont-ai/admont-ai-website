Introduction

Admont-AI is a collaborative document platform that combines Git-backed storage, hybrid search, and AI-powered content tools in a single browser-based interface.

What It Does

You write and organize documents in Markdown. Admont-AI handles everything behind the scenes:

  • Version control — Every edit is a Git commit. You can view history, compare changes, and restore any previous version.
  • Search — Find content across all your repositories using full-text search, semantic (AI-powered) search, or a combination of both.
  • AI assistant — Ask questions about your documents, generate new content, or improve existing text using integrated language models.
  • Access control — Control who can read, edit, or delete files and folders with granular, path-level permissions.
  • Multi-format editing — Edit Markdown, Mermaid diagrams, Draw.io diagrams, LaTeX, images, and plain text files from a single interface.

How It Works

Admont-AI consists of two components:

  • A Go backend that manages Git repositories, runs the search engine, connects to language models, and exposes a REST API and MCP server.
  • A React frontend that provides a rich editing experience with WYSIWYG markdown editing, a file tree sidebar, search, and an AI assistant panel.

All document content is stored in standard Git repositories. The backend clones configured repos to disk, indexes their content for search, and serves files to the frontend via the REST API. Users authenticate via OAuth 2.0 (Google or other providers), and the backend issues JWTs for session management.

Key Concepts

Concept Description
Repository A Git repository containing documents. Each repo has its own file tree, permissions, and search index.
Draft An unpublished edit. Drafts are saved separately from the Git history and published explicitly via three-way merge.
Scope When asking the AI assistant a question, you can scope it to the current page, the current repository, or all documents.
Hybrid search Combines full-text keyword matching with semantic (vector) similarity for more relevant results.
MCP Model Context Protocol — allows AI assistants like Claude to read and write wiki content programmatically.
