Quick Start

This guide walks you through getting Admont-AI running and creating your first document.

1. Start the Database

Ensure PostgreSQL is running with the pgvector extension enabled:

psql -c "CREATE DATABASE admont;"
psql -d admont -c "CREATE EXTENSION IF NOT EXISTS vector;"

Run the migrations:

psql -d admont -f internal/store/migrations/000001_create_users.up.sql
psql -d admont -f internal/store/migrations/000002_search_chunks.up.sql
psql -d admont -f internal/store/migrations/000003_ai_conversations.up.sql

2. Configure the Server

Create a minimal config.yaml:

server:
  port: 8080

database:
  url: "postgres://localhost:5432/admont?sslmode=disable"

auth:
  jwt_secret: "change-me-to-a-random-secret"
  providers:
    - name: google
      client_id: "your-google-client-id"
      client_secret: "your-google-client-secret"

cors:
  allowed_origins:
    - "http://localhost:5174"

repos:
  - slug: "my-docs"
    name: "My Documentation"
    url: "/path/to/local/git/repo"
    branch: "main"
    public_access: true

3. Start the Backend

cd admont-ai-api
go run .

The server starts on http://localhost:8080. It clones (or pulls) configured repositories on startup.

4. Start the Frontend

cd admont-ai-ui
npm install
npm run dev

Open http://localhost:5174 in your browser.

5. Log In

Click the login button in the top right. You'll be redirected to your OAuth provider (e.g., Google). After authenticating, you're redirected back with a session token.

6. Create a Document

  1. Select a repository in the sidebar.
  2. Click the + (new document) button in the sidebar toolbar.
  3. Enter a filename (e.g., hello-world.md).
  4. Write your content in the WYSIWYG editor.
  5. Click the Save button to create a draft, then Publish to commit it to Git.

7. Try the AI Assistant

Click the sparkles icon in the editor toolbar to open the AI assistant panel. You can:

  • Ask — Ask questions about the current page or search across all documents.
  • Generate — Describe content and have the AI write it directly into the editor.
  • Polish — Select text, then use quick tools to improve writing, fix spelling, or shorten.
