AI Assistant

The AI assistant helps you write, understand, and explore your documentation. It is available as a side panel in the editor and supports multi-turn conversations with context from your documents.

Opening the Assistant

Click the sparkles icon in the editor toolbar (or the "What is this page about?" prompt) to open the AI assistant panel on the right side of the screen.

Modes

Ask

Ask questions about your documents. The assistant uses RAG (Retrieval-Augmented Generation) to search relevant content and provide grounded answers with source citations.

Type a question and press Enter. The assistant searches your documents, retrieves relevant excerpts, and generates an answer. Sources are listed below each response — click a source to navigate to that document.

Scope Selector

Control what context the AI uses when answering:

Scope Behavior
This page Uses only the current page content (no document search)
This repo Searches within the current repository
All docs Searches across all accessible repositories

The scope selector is a dropdown next to the model selector at the bottom of the panel.

Generate

Describe content you want and the AI writes it directly into the editor at the cursor position. Useful for creating new sections, boilerplate, or structured content.

Polish (Improve Writing)

Select text in the editor, then use the polish tools:

Tool Description
Improve writing Rewrites the selection for clarity and flow
Fix spelling & grammar Corrects errors without changing meaning
Shorten Condenses the text while keeping key information
Summarize Generates a summary of the selected text
Explain Provides an explanation of the selected text

Polish tools replace the selected text in the editor with the improved version.

Conversations

The AI assistant supports multi-turn conversations. Follow-up questions retain context from earlier messages in the same conversation.

Conversation Management

  • New conversation — Click the + button or select "New conversation" from the dropdown.
  • Switch conversations — Click the conversation dropdown in the panel header to see recent conversations and switch between them.
  • Delete conversations — Hover over a conversation in the dropdown and click the trash icon.

Conversations are persisted server-side and available across browser sessions.

Auto-Summarization

When a conversation exceeds 20 messages, older messages are automatically summarized by the AI to keep token usage manageable. A "Earlier conversation summarized" divider appears where summarization occurred. Recent messages (the last 6) are always kept in full.

Model Selection

Use the model dropdown at the bottom of the panel to choose which language model to use. Available models depend on which LLM providers are configured by the administrator. Your selection is saved in the browser and persists across sessions.

Supported providers: Anthropic, OpenAI, Google, DeepSeek, Meta, Mistral, Ollama, Perplexity, xAI.

Token Usage

Each assistant response shows token counts: input tokens (sent to the model) and output tokens (generated by the model), displayed as ↑ and ↓ respectively.
